## ğŸš€ How to Use

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd DSA
   ```

2. **Navigate to the desired language folder** (Java/, JS/, python/)

3. **Choose a topic folder** (array/, string/, etc.)

4. **Run the specific problem file**:
   - For JavaScript: `node filename.js`
   - For Java: Compile and run `javac filename.java && java filename`
   - For Python: `python filename.py`

5. **Review the code** and understand the algorithm implementation
